⏱️ Stopwatch Application

A fully functional stopwatch web application with start, pause, reset, and lap tracking features. Includes keyboard shortcuts for power users.

📌 Live Demo

View Live Demo at ""

🎯 Features

1. Start/Pause/Reset: Core stopwatch controls
2. Lap Trackin: Record and display lap times while the stopwatch runs
3. Millisecond Precision: Updates every 10 milliseconds
4. Keyboard Shortcuts: Space (Start/Pause), L (Lap), R (Reset)
5. Responsive Design: Works on all screen sizes

🛠️ Technologies Used

| Technology | Purpose 
|------------|---------
| HTML5      | Structure 
| CSS3       | Styling, animations 
| JavaScript | Timer logic, DOM manipulation, event handling
